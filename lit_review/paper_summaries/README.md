# Paper Summaries

This directory stores AI-generated markdown summaries of papers in `lit_review/papers/`, produced by the `/create-paper-summary` skill.

## Contents

Summaries are added here as they are generated. The subdirectory structure should mirror that of `papers/`.

## Other instructions

- Summary files are named by BibTeX key (e.g., `smith2023social.md`).
- Each summary is generated from its corresponding PDF using the `/create-paper-summary` skill.
