# Papers

This directory stores raw PDF files collected during the literature review phase, organized into subdirectories by topic or category.

## Contents

Add subdirectories here as needed to organize papers by topic or category (e.g., `methods/`, `background/`).

## Other instructions

- PDFs are excluded from version control via `.gitignore`; only the directory structure and `.gitkeep` files are tracked.
- Name files using the convention: `Author - YYYY - Title.pdf` (e.g., `Smith - 2023 - Social Media Effects.pdf`).
